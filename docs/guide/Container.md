# 自定义容器

代码：
```
    ::: tip 提示
    this is a tip
    :::

    ::: warning 注意
    this is a tip
    :::

    ::: danger 警告
    this is a tip
    :::

    ::: details
    This is a details block, which does not work in Internet Explorer or Edge.
    :::
    
```

效果：
::: tip 提示
this is a tip
:::

::: warning 注意
this is a tip
:::

::: danger 警告
this is a tip
:::

::: details
This is a details block, which does not work in Internet Explorer or Edge.
:::
