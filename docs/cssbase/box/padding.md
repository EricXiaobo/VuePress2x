# padding

- 四值法

| 属性值  | 说明                     |
| ------- | ------------------------ |
| padding | 上、右、下、左（顺时针） |

```css
padding: 15px 26px 39px 54px;
```

- 三值法

| 属性值  | 说明                   |
| ------- | ---------------------- |
| padding | 上、左右、下（顺时针） |

```css
padding: 15px 26px 54px;
```

- 二值法

| 属性值  | 说明         |
| ------- | ------------ |

- 一值法

| 属性值  | 说明                  |
| ------- | --------------------- |
| padding | 20%（四个边都是 20%） |

<!-- ![](/images/css/004.png) -->

```css
padding: 50px;
```

<img src="/images/css/004.png" style="width: 50%; display: block; margin: 0 auto;">

- 四参数

| 属性值         | 说明 | 类似标签 |
| -------------- | ---- | -------- |
| padding‐top    |      |          |
| padding‐left   |      |          |
| padding‐right  |      |          |
| padding‐bottom |      |          |

```css
padding-top: 15px;
padding-right: 30px;
padding-bottom: 25px;
padding-left: 49px;
```

<img src="/images/css/006.png" style="width: 50%; display: block; margin: 0 auto;">

- 另类写法

```css
padding: 50px; //设置四边
padding-bottom: 25px; //再把某一边重置设置
```
