# width

| 属性值 |                  说明                   | 类似标签 |
| ------ | :-------------------------------------: | -------: |
| auto   |                  默认                   |          |
| px     |                                         |          |
| %      | 定义参考父元素宽度 width 的百分比宽度。 |          |

::: tip 提示
`div` `body`标签的属性默认值是 auto。

`div`标签需独占一行，根据父级元素自动撑满一行。

`span`标签不需独占一行，根据自身内部元素自动撑满。
:::
