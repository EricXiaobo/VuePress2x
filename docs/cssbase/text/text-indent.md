# text-indent
text-indent 文本缩进

| 属性值 |                       说明                        | 类似标签 |
| ------ | :-----------------------------------------------: | -------: |
| px     |                首行缩进所少个像素                 |          |
| em     |              首行缩进所少个中文字符               |          |
| 百分比 | 缩进文字所在标签的父级标签的 width 属性值的百分比 |          |

::: tip 提示
em 值：向左为负缩进，向右为正缩进
:::

```css
.none {
  text-indent: -50%;
}
```
