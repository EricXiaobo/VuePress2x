# text-decoration
text-decoration 文本修饰

| 属性值       |      说明      |         类似标签 |
| ------------ | :------------: | ---------------: |
| none         |  木有（默认）  |                  |
| overline     |     上划线     |                  |
| line-through | 中划线，删除线 | del 标签的默认值 |
| underline    |     下划线     |   a 标签的默认值 |
