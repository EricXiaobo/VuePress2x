# ECMAScript 概述

- 通常将 ECMAScript 看作是 JavaScript 的标准规范。实际上 JavaScript 是 ECMAScript 的扩展语言。

- ECMAScript 只是提供了最基本的语法。JavaScript 在语言基础上进行了扩展
