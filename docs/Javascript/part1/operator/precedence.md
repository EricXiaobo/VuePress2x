# 运算优先级

| 运算符     | 属性            |
| ---------- | --------------- |
| `()`         | 优先级最高      |
| 一元运算符 | `++ -- ！`        |
| 算术运算符 | `先\* / % 后 + -` |
| 关系运算符 | `> >= < <=`       |
| 相等运算符 | `== != === ! ==`  |
| 逻辑运算符 | 先`&&` 后 &vert;&vert;  |
| 赋值运算符 |                 |

```js
// () → 一元 → 算术 → 比较 → 逻辑 → 赋值
var a = 4;
var num = (1 * (2 + 3) && ++a) || (5 > 6 && 7 < 8) || !9;
// num = (1 * 5 && ++a) || (5 > 6 && 7 < 8) || !9;
// num = (1 * 5 && 5) || (5 > 6 && 7 < 8) || false;
// num = (5 && 5) || (5 > 6 && 7 < 8) || false;
// num = (5 && 5) || (false && true) || false;
// num = 5 || false || false;
// num = 5;
console.log(num);
```
