# prompt() 语句

- prompt：提示。
- 作用：弹出一个对话框，内部有一个提示语句以及一个输入框，可以在输入框中根据提示任意输入内容。
- prompt 语句也是 js 内置的一个功能，必须加小括号执行，有两个参数可以进行传递，每个参数一般都是字符串类型，必须加引号，两个参数中间用逗号分隔，引号和逗号都必须是英文输入法状态。

## 语法

```html
<head>
  <script>
    prompt("请输入您的年龄","18");
  </script>
</head>
```
