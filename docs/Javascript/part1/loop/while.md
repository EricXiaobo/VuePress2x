# while 循环

- while 循环是一种前测试循环语句，在执行循环体之前都要测试入口条件，条件为真继续执行，条件为假直接跳出循环。

## 语法

```js
    while(条件表达式){
         循环体；
    }
```

- 当条件表达式为真时，执行循环体，如果为假，跳出循环。

```js
// 输出1到10之间的数字
var i = 1;
while (i <= 10) {
  i++;
  console.log(i);
}
```

## 注意事项

- 如果需要循环变量参与，必须定义在循环外部，避免被重置。
- 循环变量自加的过程写在循环体内部。

## 总结

- 遇到必须先执行一次结构体的循环，使用 do while 循环。
- 先测试入口条件的循环，可以选择 for 循环或 while 循环。
