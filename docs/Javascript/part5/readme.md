# JQuery 快速网页交互开发

## 版本发展
- 1.X：兼容 IE6/7/8，是工作中最常使用的，学习 1.12 版本。
- 2.X：不兼容 IE6/7/8，多用于 jQuery 官方调整 bug 使用。工作中不使用。
- 3.X：不兼容 IE6/7/8，只能在高版本浏览器中使用，是现在 jQuery 官方主要的维护升级的
版本。
