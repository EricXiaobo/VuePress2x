# 函数表达式

- 函数表达式是函数定义的另外一种方式。
- 定义方法：就是将函数的定义、匿名函数赋值给一个变量。

* 函数定义赋值给一个变量，相当于将函数整体矮化成了一个表达式。

```js
// 定义一个函数表达式
var foo = function fun() {
  console.log(1);
};
```

- 匿名函数：函数没有函数名。

```js
var foo2 = function() {
  console.log(2);
};
```

- 调用函数表达式，方法是给变量名加 执行，不能使用函数名加 执行。

```js
// 调用函数式，只能用变量名调用，函数名调用不成功
// fun();
foo();
foo2();
```
