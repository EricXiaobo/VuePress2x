# 跨域请求问题和同源策略

## 同源策略
- 同源策略是浏览器的一种安全策略，所谓同源是指域名，协议，端口完全相同
- 在同源策略下，只有同源的地址才可以相互通过 AJAX 的方式请求。
- 同源或者不同源说的是两个地址之间的关系，不同源地址之间请求我们称之为跨域请求
